Installation Guide

This is a quick guide on installation, and updating. Please read the following steps carefully. Its really help you alot.

 

1.1 Plugin Installation

To install Motif WooCommerce Feature Video plugin you must have a working and latest version of WordPress and WooCommerce plugin it self already installed. For information on installing the WordPress, please see the WordPress Codex - How to Installing WordPress. To installing the WooCommerce and configure the shop please visit the following link Installing WooCommerce.

Now the next step is the Motif WooCommerce Feature Video installation. In the full .ZIP archive you have downloaded from Codecanyon.net you will find a file called motif-feature-video.zip. This file is the installable .ZIP archive you can install via WordPress.

Now is time to install Motif WooCommerce Feature Video plugin please read all below steps carefully.

 

Installing from WordPress

Follow the steps bellow to install from WordPress Admin Dashboard

  1. Navigate to Plugins
  2. Click Install Plugins and click Add New
  3. Click Upload Plugin button
  4. Navigate motif-feature-video.zip on your computer and click Install Now
  5. When installation is done please click Activate to activate plugin

Installing using FTP

Follow the steps bellow to install using FTP

  • Unzip motif-feature-video.zip to a desired location on your computer. These extracted files will be uploaded later via FTP
  • Connect to your server via FTP (Use FileZilla or any other FTP client)
  • Navigate to /wp-content/plugins/ server directory
  • Locate the local directory where you have unzipped the motif-feature-video.zip archive
  • Upload local folder /motif-feature-video/ to remote /wp-content/plugins/
  • To activate the newly installed plugin navigate to Plugins and click the Activate button within the Motif WooCommerce Feature Video Plugin.

1.2 Plugin Update

Updating from WordPress Admin Panel

Follow all the important steps to update from WordPress Admin Dashboard

  • Navigate to Plugins
  • Find the currently installed Motif WooCommerce Feature Video Plugin and click Deactivate
  • After deactivation click Delete to remove the old version (your saved setting will not be lost)
  • Click Install Plugins and click Add New
  • Click the Upload Plugin button
  • Navigate the new version archive motif-feature-video.zip on your computer and click Install Now
  • When installation is done please click Activate to activate plugin

Updating Via FTP

Follow the steps bellow to update using FTP

  • Unzip the new version archive motif-feature-video.zip to a desired location on your computer. These extracted files will be uploaded later via FTP
  • Connect to your server via FTP (Use CuteFTP or any other FTP client)
  • Navigate to /wp-content/plugins/ server directory and delete the current plugin folder /motif-feature-video/
  • Locate the local directory where you have unzipped the new version archive motif-feature-video.zip
  • Upload local folder /motif-feature-video/ to remote /wp-content/plugins/

How to Configure the Plugin

At the backend, go to "Setting Option". Here you will find the following settings:

Youtube/Vimeo Settings Options:

  1. Control of videos: This is the control of videos like progress bar, full screen option etx
  2. Height of video Frame: This is height of video frame on single page
  3. Width of video Frame: This is Width of video frame on single page
  4. Related Video: Can be the related videos auto load after the video end
  5. Video Title: For the video title hide or show
  6. Video Auto Play: Video can auto play once a user can vsit the product page
